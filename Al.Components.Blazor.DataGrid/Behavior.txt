1. Изменение ширины столбца.
1.1. Если изменение доступно для столбца, то справа внутри столбца появляется поле для захвата края столбца.
1.2. При недоступности изменения ширины для столбца поле для захвата не отображается.
1.3. При наведении на поле для захвата края столбца меняется курсор.
1.3. При измененении ширины меняются colspan-ы на заголовках и на теле таблицы
2.
2. Перестановка столбцов
2.1. Если перестановка столбцов доступна для грида, то при наведении на заголовок столбца меняется курсор.
2.2. Если не доступна, то столбцы двигать нельзя и курсор обычный
2.3. При захвате столбца он меняет внешний вид
2.4. При переносе столбца на ближайший невозможно выбрать позицию, в которой он встанет в то же место.
2.5. При переносе столбца на другом столбце появляетс область, означающая, что столбец будет установлен до или после столбца.
2.6. При отпускании столбца на другом месте:
2.6.1. Изменяются colspan-ы на заголовках и на теле таблицы
2.6.2. Появляется лоадер
2.6.3. Обновляются данные в теле таблицы
3. Сортировка столбца
1.1. Если сортировка доступна для столбца, то внутри заголовка справа появляется область индикатора сортировки
1.2. При наведении на заголовок столбца меняется курсор и позволяет кликнуть по нему
1.3. По клику:
1.3.1. меняется индикатор сортировки: стрелка вверх, стрелка вниз, пусто.
1.3.2. Появляется лоадер
1.3.3. Обновляются данные в теле таблицы
